# 行为准则

为构建友好、包容的社区，我们遵循 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)（英文原文）。以下为关键要点摘要：

## 期望的行为
- 以尊重、体贴的态度对待所有参与者。
- 积极接纳不同观点，及时沟通并解决分歧。
- 主动给予和接受建设性反馈。
- 与社区成员合作，共同创造高质量的解决方案。

## 不可接受的行为
- 使用带有歧视、骚扰或攻击性的言语与图像。
- 有意恐吓、跟踪、辱骂或羞辱他人。
- 公开他人的隐私信息。
- 其他被视为不专业或令人反感的行为。

## 责任与处理
- 项目维护者有权力与责任删除、编辑或拒绝与本准则不一致的评论、提交、Wiki 编辑、Issue 和其他贡献。
- 若发现违规行为，可通过 Issue 或邮件（例如 `maintainers@codex-switch.dev`）联系维护团队。维护者将及时评估并采取措施。

感谢每一位遵守准则、维护社区环境的贡献者！
